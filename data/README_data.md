# 📂 data/

A pasta `data/` contém os documentos utilizados como base de conhecimento para o sistema de RAG (Retrieval-Augmented Generation).

Esses arquivos fornecem o contexto necessário para a geração dos Termos de Referência, garantindo maior precisão e aderência às normas e práticas institucionais.

## Estrutura

- `data/laws/`
  - Contém leis, normas e regulamentos (ex: Lei 14.133/2021)
  - Os documentos são processados e divididos por artigos
  - Utilizados como base legal para fundamentação das seções

- `data/tr_models/`
  - Contém modelos institucionais de Termos de Referência
  - Os documentos são segmentados por seções
  - Servem como referência estrutural e de linguagem

## Observação

⚠️ Os arquivos reais não estão versionados no repositório por conterem informações sensíveis.

Foram incluídos apenas exemplos fictícios para demonstrar a estrutura esperada.
